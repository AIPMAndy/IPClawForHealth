# 💰 变现话术库

> 成交、催单、答疑的话术模板，让变现更自然

## 使用前先记住

话术的作用不是“逼单”，而是帮助用户更快理解自己是否适合你的服务。

在健康行业，成交表达尤其要避免：

- 夸张承诺
- 疗效式说法
- 焦虑式施压
- 无边界地替代专业判断

---

## 一、信任建立类

### 破冰开场

```
"我看你关注我{时间}了，今天终于等到你提问了"
```

```
"感谢你的信任，能问出这个问题，说明你已经想清楚了"
```

### 展示专业

```
"我在这行做了{年数}年，服务过{人数}+学员，
最大的感受是：{洞察}"
```

```
"这个问题问得好，我带过的学员里，90%都遇到过"
```

---

## 二、价值传递类

### 介绍服务

```
"我们的{产品}，主要是帮你解决{问题}，
和市面上不一样的是：{差异化}"
```

```
"很多{用户}问我这个课到底有什么用，
简单说三点：
第一，{价值1}
第二，{价值2}
第三，{价值3}"
```

### 成果展示

```
"跟着我{时间}的学员，{比例}%都{成果}：
比如{案例1}，{案例2}，{案例3}"
```

---

## 三、成交催单类

### 限时优惠

```
"这个价格{只剩/只收}到{时间}，
因为{原因}，之后{恢复}"
```

```
"今天报名额外送{赠品}，这个赠品{价值}，只给今天"
```

### 稀缺紧迫

```
"这一期{名额}只收{人数}，现在还剩{数量}"
```

```
"我每个月{只带/只服务}{人数}，多了顾不过来"
```

### 消除顾虑

```
"你担心{顾虑}，完全理解。
{证据1}、{证据2}，而且{保障}"
```

```
"之前学员也担心这个问题，后来{案例}，
他说{反馈}"
```

---

## 四、应对异议类

### 太贵了

```
"确实不便宜，但你知道{算出价值}吗？
而且{投资回报}，{对比}"
```

```
"贵是因为{原因}，便宜的产品{后果}，
你会选哪个？"
```

### 再考虑考虑

```
"考虑是应该的，这是大事。
但我想问你：{考虑的具体是什么}？
如果{问题解决}，你会决定吗？"
```

```
"考虑清楚是对的。
这样，我给你{免费/小成本}试一下，
体验完再决定"
```

### 暂时不需要

```
"完全理解，很多人{之前}也是这么说，
但后来{结果}，所以{观点}"
```

## 五、追销复购类

### 追加销售

```
"既然你已经{完成}，接下来{升级}会更有效"
```


## 📌 使用技巧

### 原则1：先给后卖

```
先帮助 → 建立信任 → 自然成交
```

### 原则2：少即是多

```
话术在精不在多
背3个模板 > 背30个
```

### 原则3：真诚最重要

```
话术是框架，真诚是灵魂
复制话术，注入真诚
```

### 原则4：先说边界，再谈成交

```text
适合谁 / 不适合谁 / 需要什么配合 / 预期是什么
```

## 🛡️ 合规表达替换

| 不建议直接说 | 更建议这样说 |
|-------------|--------------|
| 跟着我一定瘦 | 我会帮你搭建更容易坚持的减脂执行方案 |
| 这个方法所有人都能用 | 这套方法更适合某类人群，具体还要看你的情况 |
| 不用去医院 | 如果你有明显不适或基础疾病，建议同步咨询医生 |
| 这期必须报名 | 如果你目前正想系统解决这个问题，这期可能适合你 |
