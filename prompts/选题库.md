## 💡 使用技巧

### 选题公式

```
爆款选题 = 痛点 + 数字 + 悬念/利益
```

### 优化技巧

1. **加数字**：具体可信
2. **加悬念**：引发好奇
3. **加利益**：明确好处
4. **加身份**：增加权威
5. **加情绪**：引发共鸣

### 拍摄角度

| 痛点类型 | 适合角度 |
|----------|----------|
| 知识科普 | "终于有人说清楚了" |
| 方法干货 | "附详细方案" |
| 避坑指南 | "90%的人都踩过" |
| 认知颠覆 | "你可能一直做错了" |
| 经验分享 | "我带过的学员..." |

## 🔥 快速出片模板

### 模板1：痛点引入式

```
[开头3秒]
"你是不是也有这种感觉：{痛点描述}"

[主体]
"其实{原因分析}，我{经验/方法}"

[结尾]
"如果你也有{问题}，评论告诉我"
```

### 模板2：颠覆认知式

```
[开头]
"很多人觉得{常识}，但实际上{颠覆观点}"

[论证]
"{原因1}、{原因2}、{原因3}"

[结尾]
"转发给需要的朋友"
```

### 模板3：案例故事式

```
[开头]
"前两天有个{用户}，{问题}..."

[过程]
"我教他{方法}，结果{结果}"

[结尾]
"如果你也想{好处}，点击下方..."
```
