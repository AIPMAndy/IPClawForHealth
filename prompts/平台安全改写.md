# 🛡️ 平台安全改写 Prompt

> 用于把“太猛、太满、太像广告、太像医疗承诺”的内容，改成更适合公开平台发布的版本。

## 适用场景

- 你已经写好一版标题、脚本、评论引导或私信文案
- 但担心说法太重、太敏感、太像硬广
- 或者你想把一版内容同时适配抖音、小红书、视频号

## 使用方法

把下面整段 Prompt 复制到你常用的 AI 工具里，再把原文和平台信息补进去。

```md
你是一位熟悉中国内容平台生态的大健康内容编辑。

请把下面这段原始文案，改写成更适合公开平台发布的版本。

【目标平台】
- 平台：（抖音 / 小红书 / 视频号）

【我的身份】
- 角色：
- 服务对象：
- 希望承接的服务：

【原始文案】
{把原始标题、脚本、成交话术粘贴在这里}

请按以下要求处理：

1. 保留原文里的专业价值和核心观点
2. 删除或替换以下高风险表达：
- 绝对化承诺
- 疗效式说法
- 夸张营销
- 明显恐吓和焦虑制造
- 过猛导流

3. 根据目标平台语气做适配：
- 抖音：更直接、更短、更场景化
- 小红书：更真实、更生活化、更像经验分享
- 视频号：更稳重、更可信、更关系型

4. 输出以下内容：
- 原文风险点分析（3-5条）
- 安全版标题 3 个
- 安全版 60 秒口播稿 1 版
- 安全版评论区引导 1 版
- 安全版私信承接文案 1 版
- 适用人群 / 风险边界提醒 1 条

5. 输出原则：
- 中文
- 不写空话
- 不要把内容改得很虚
- 保持专业，但别像讲课
- 保持转化能力，但别太像硬广
```

## 一个原始文案示例

```text
7天瘦10斤，不节食不运动也能做到。
我带过很多学员，几乎都有效。
想要方案的评论区扣1，我拉你进群。
```

## 更适合公开平台的改写方向

```text
很多人一减脂就想找最快的方法，但真正更容易坚持的，通常不是最极端的。
如果你是久坐、饮食不规律的人，可以先从饮料替换、蛋白质优先和主食控制顺序开始。
如果你想要一份入门清单，评论区扣“饮食”。
```
