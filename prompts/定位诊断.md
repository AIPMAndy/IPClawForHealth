# 🎯 创始人口播 IP 定位诊断

> 用 AI 帮助大健康行业创始人找到自己的 IP 定位

## 使用方法

将以下 prompt 粘贴到 ChatGPT / Claude / Kimi 等 AI 对话工具中，按提示回答问题即可获得你的 IP 定位报告。

---

## Step 1: 基础信息

请回答以下问题：

1. 你的身份是？（如：健身教练、营养师、健康管理师、中医、瑜伽老师等）
2. 你从业多少年？有什么专业背景？
3. 你主要服务哪类用户？（如：职场人群、宝妈、老年人、亚健康人群等）
4. 你的核心优势是什么？（如：专业资质、实战经验、独特方法论等）
5. 你目前有做自媒体吗？效果如何？
6. 你希望通过口播IP实现什么目标？（如：获取客户、品牌建设、知识变现等）

---

## Step 2: AI 诊断 Prompt

将你的答案粘贴到以下 prompt 中：

```
你是 IP 定位专家，帮助大健康行业创始人找到个人品牌定位。请根据以下信息生成一份 IP 定位报告：

【基本信息】
身份：{你的身份}
从业年限：{年限}
专业背景：{背景}
目标用户：{用户群体}
核心优势：{优势}
自媒体现状：{现状}
目标：{目标}

请输出：

## 一、IP 定位陈述（100字内）
一句话说明你是谁、解决什么问题、有什么独特优势

## 二、目标用户画像
- 人口特征
- 痛点需求
- 价值观

## 三、内容方向建议
- 3个主要话题领域
- 每个领域3个具体选题

## 四、差异化定位
与同类竞品的区别是什么

## 五、第一步行动建议
接下来7天应该做什么
```

---

## 📋 定位诊断示例

### 案例：健身教练

**IP 定位陈述**：
> "让30岁职场人用碎片时间练出健康身材的运动规划师"

**目标用户**：
- 25-40岁职场人群
- 久坐、亚健康
- 没时间去健身房

**内容方向**：
1. 碎片化健身教程
2. 办公室拉伸指南
3. 饮食与运动结合

---

## 🚀 立即开始

1. 回答 Step 1 的问题（5分钟）
2. 复制到 AI 工具运行（2分钟）
3. 获得你的专属 IP 定位报告
