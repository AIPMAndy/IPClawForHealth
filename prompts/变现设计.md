# 💰 大健康行业创始人 IP 变现设计 Prompt

> 用于把「内容影响力」进一步整理成「产品梯度 + 转化路径 + 成交动作」。

## 适用场景

- 已经开始做短视频，但不知道怎么承接咨询和成交
- 有一定专业能力，却没有清晰的产品阶梯
- 想把内容、私域、咨询、课程、训练营串成一条线

## 使用方式

把下面整段 Prompt 复制到你常用的 AI 工具里，然后把括号里的内容替换成你自己的信息。

```md
你是一位擅长服务型生意和创始人 IP 变现设计的顾问。

请基于以下信息，帮我设计一套适合大健康行业创始人的变现路径：

【我的身份】
- 行业/赛道：（如：健身、营养、中医、康复、睡眠、心理健康）
- 当前角色：（如：创始人、医生、教练、馆主、顾问）
- 我的专业优势：（列 3 条）
- 我的典型客户：（列 3 类）

【我现在的内容基础】
- 平台：（抖音 / 视频号 / 小红书 / 公众号 / 直播）
- 当前内容主题：
- 当前粉丝规模：
- 当前私域规模：
- 当前最容易成交的服务：

【我的业务目标】
- 希望主推的高价值服务：
- 期望客单价区间：
- 期望 3 个月内达成的营收目标：
- 我能接受的交付方式：（1v1 / 小班 / 录播 / 线下 / 社群）

请输出以下内容：

1. 一句话价值主张
- 用「我帮助谁，通过什么方法，获得什么结果」格式写

2. 产品梯度设计
- 引流产品
- 低价试用产品
- 核心成交产品
- 高客单长期服务
- 每个产品分别写清：适合谁、交付形式、建议价格、成交理由

3. 内容到成交的转化路径
- 公域内容怎么引流
- 私域第一步聊什么
- 第二步怎么筛选高意向用户
- 第三步怎么推进成交

4. 成交话术框架
- 首次私信/加微后的欢迎语
- 需求诊断的 5 个关键问题
- 报价前的价值铺垫
- 报价后的异议处理

5. 30 天变现动作计划
- 每周重点
- 每周内容主题
- 每周转化动作
- 每周复盘指标

6. 风险提醒
- 我这个赛道容易踩的 3 个坑
- 哪些承诺不能乱说
- 哪些表达需要更专业、更合规

输出要求：
- 中文
- 强执行导向
- 少讲空话，多给表格和清单
- 结果要能直接拿去落地
```

## 推荐搭配

1. 先用 `定位诊断.md` 明确你要服务的细分人群
2. 再用 `选题库.md` 确定引流内容方向
3. 先看 `docs/健康内容合规清单.md`，确认你的表达边界
4. 用本文件设计你的产品和转化路径
5. 最后再回到 `话术库.md` 和 `脚本模板.md` 打磨实际表达

## 你最终要拿到的结果

- 一句话定位
- 4 层产品梯度
- 内容到成交的动作链
- 可直接复制的成交话术
- 30 天变现执行表

## ⚠️ 变现设计提醒

健康行业的成交设计，尽量卖“过程支持、执行陪伴、方法框架、内容服务”，少卖“确定结果”。

例如：

- 更适合卖：评估表、饮食清单、训练计划、打卡营、陪跑服务
- 尽量少直接承诺：包瘦、包治、包改善、人人见效
