## 💡 使用场景

### 场景1：刚起步的健身教练

> "我不知道怎么做IP，感觉自己不够专业"

**解决方案**：
1. 用定位诊断找到定位
2. 从选题库选"碎片化健身"方向
3. 用脚本模板写"5分钟练出马甲线"
4. 30分钟出片发布

### 场景2：想转型的营养师

> "我想线上获客，但不知道写什么"

**解决方案**：
1. 定位"科学饮食科普"
2. 选题库选"辟谣"系列
3. 脚本模板写"这5种伪健康食物"
4. 持续输出建立专业信任

### 场景3：养生馆老板

> "年轻人不相信传统养生"

**解决方案**：
1. 定位"现代人听得懂的养生"
2. 用年轻人语言讲古老智慧
3. 线上内容 → 线下体验闭环
