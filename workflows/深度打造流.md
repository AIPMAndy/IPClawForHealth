# 🗓️ 系统性IP打造工作流（30天）

> 目标：30天从0到1建立创始人口播IP

---

## 📅 第1周：定位与准备

### Day 1-2：IP定位

| 任务 | 输出 |
|------|------|
| 1. 完成 `prompts/定位诊断.md` | IP定位陈述 |
| 2. 分析3-5个对标账号 | 竞品分析 |
| 3. 确定内容方向 | 内容系列规划 |

### Day 3-4：基础设施

| 任务 | 输出 |
|------|------|
| 1. 注册视频账号 | 账号设置 |
| 2. 准备设备 | 手机+支架 |
| 3. 布置拍摄环境 | 背景+光线 |
| 4. 设计头像+封面 | 视觉统一 |

### Day 5-7：内容储备

| 任务 | 输出 |
|------|------|
| 1. 从选题库选10个选题 | 选题清单 |
| 2. 写3条脚本 | 脚本库 |
| 3. 做1次合规检查 | 风险词清单 |
| 4. 拍3条视频 | 素材 |

---

## 📅 第2周：内容启动

### Day 8-10：日更测试

- 每天1条视频
- 测试不同选题、数据
- 记录每条数据

### Day 11-14：数据分析

| 指标 | 分析 |
|------|------|
| 完播率 | 哪个选题完播最高 |
| 点赞率 | 什么内容互动好 |
| 关注率 | 哪个类型转化关注 |

### 输出

- 数据分析报告
- 优化内容方向
- 确定2-3个主要内容系列

---

## 📅 第3周：内容优化

### Day 15-18：质量提升

- 脚本优化：更口语化、有钩子
- 拍摄优化：表情、语速、眼神
- 剪辑优化：节奏、字幕、配乐

### Day 19-21：互动运营

- 回复每条评论
- 引导评论互动
- 私域转化测试

---

## 📅 第4周：变现探索

### Day 22-24：产品设计

| 层级 | 产品 | 价格 |
|------|------|------|
| 引流 | 资料包/电子书 | 0-9.9元 |
| 低价 | 录播课/社群 | 99-399元 |
| 高价 | 私教/陪跑 | 1000+元 |

### Day 25-28：变现测试

- 发布变现内容
- 引导私信
- 成交测试

### Day 29-30：复盘总结

- 数据复盘
- 问题梳理
- 下月规划

## 🔥 关键成功要素

### 1. 定位清晰

```
一句话能说清：你是谁、解决什么问题
```

### 2. 内容聚焦

```
一个细分领域 + 2-3个内容系列
```

### 3. 持续输出

```
日更是起步，周更是底线
```

### 4. 数据驱动

```
用数据指导内容方向
```

### 5. 及时变现

```
从第一天就考虑变现
```

## ⚠️ 常见问题

### Q：不知道拍什么

```
从选题库选，坚持发
```

### Q：数据不好

```
分析数据，找最好的选题方向
优化脚本、拍摄、剪辑
```

### Q：不敢出镜

```
先拍背影、音频
逐渐过渡到出镜
```

### Q：不会变现

```
先从免费资料开始
测试用户付费意愿
```
