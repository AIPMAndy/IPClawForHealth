# 🤝 贡献指南

感谢你对 IPClawForHealth 的兴趣！

## 如何贡献

### 1. 贡献选题

- 添加新的行业选题到 `prompts/选题库.md`
- 格式：`| 选题 | 方向 | 痛点 |`

### 2. 贡献案例

- 添加你的成功案例到 `examples/案例库.md`
- 包括：背景、策略、成果、关键动作

### 3. 贡献话术

- 添加变现话术到 `prompts/话术库.md`
- 包括：场景、话术、适用条件

### 4. 提交 Issue

- 发现 bug？请提交 Issue
- 有新功能建议？请提交 Issue

### 5. Pull Request

1. Fork 本项目
2. 创建新分支 (`git checkout -b feature/xxx`)
3. 提交更改 (`git commit -m 'Add xxx'`)
4. 推送到分支 (`git push origin feature/xxx`)
5. 提交 Pull Request
