# ⚖️ 健康内容合规清单

> 适用于大健康行业创始人、营养师、教练、中医/养生从业者在做短视频、直播、私域沟通和产品销售时的基础表达自检。

## 先说清楚

- 本清单是内容表达层面的风险提醒，不构成法律意见或医疗建议。
- 涉及疾病诊断、治疗方案、处方建议、药品功效、医疗器械宣传时，优先咨询专业律师或合规顾问。
- 你发布的是“内容”，不是“诊疗结论”；你提供的是“信息和方法”，不是“保证结果”。

## 高风险表达，尽量不要直接说

以下表达在健康行业尤其容易出问题，默认不要直接用于公开内容、成交页、海报、私信模板：

- “包瘦”“7天必瘦”“不瘦退款”
- “治愈”“根治”“彻底解决”
- “绝对安全”“零副作用”“人人适用”
- “只要照做一定有效”
- “替代医院/替代医生/替代药物”
- “这个方法谁都能用”
- “学完立刻月入多少”
- “这是唯一有效的方法”

## 更稳妥的替代表达

| 高风险说法 | 更稳妥的说法 |
|-----------|--------------|
| 保证瘦 10 斤 | 帮你建立更容易坚持的减脂行动方案 |
| 治好失眠 | 帮你改善睡前习惯与睡眠管理流程 |
| 彻底调理好 | 帮你从作息、饮食、训练等方面做系统调整 |
| 百分百有效 | 对一部分人更适合，建议结合个人情况判断 |
| 人人都能做 | 更适合哪类人群，不适合哪类人群 |
| 不用看医生 | 如有明显症状或基础疾病，建议同步咨询医生 |

## 发布内容前，至少检查这 6 件事

1. 有没有把“经验分享”说成“医疗结论”。
2. 有没有把“个体案例”说成“普遍结果”。
3. 有没有出现绝对化承诺。
4. 有没有省略“不适合人群 / 使用前提 / 风险提醒”。
5. 有没有把他人案例、资质、数据说得像公开可验证背书，但其实拿不出来源。
6. 有没有在成交内容里用过强焦虑和恐吓式表达逼单。

## 不同内容类型的额外提醒

### 1. 科普类内容

- 优先用“适合 / 可能 / 建议 / 常见情况”这类表述。
- 涉及营养、康复、体检、慢病、心理健康时，主动补一行适用边界。
- 如果引用结论，尽量注明来源类型，如“指南建议”“临床常见建议”“公开科普共识”。

### 2. 案例类内容

- 最好标明“教学示意案例”“经授权分享”或“已做匿名处理”。
- 不要把单一案例结果包装成普遍承诺。
- 使用前后对比时，避免暗示“所有人都能得到同样结果”。

### 3. 成交类内容

- 少用“错过就亏”“不买就会怎样”的压迫式催单。
- 多用“适合谁 / 不适合谁 / 你当前最该先做什么”。
- 如果服务效果高度依赖配合度，明确说出来。

### 4. 直播 / 私信答疑

- 不直接替代正式问诊、检查、处方。
- 对明显超出自己专业边界的问题，直接建议线下或专业机构处理。
- 涉及孕妇、儿童、老年慢病、术后恢复等人群时，边界说得更明确。

## 一个简单好用的表达框架

```text
先说适合谁
再说为什么
再说可以怎么做
最后补一句风险边界或建议
```

示例：

```text
如果你是久坐、睡眠不稳定、但没有明确器质性疾病的人，
可以先从固定起床时间和减少晚间刺激开始。
这类方法更适合生活方式调整阶段的人。
如果你已经长期失眠或伴随明显不适，建议及时就医评估。
```

- [ ] 写清了适用人群

- [ ] 数据、案例、资质说法经得起追问
- [ ] 成交文案没有过度制造焦虑
